Chatwright greetbot — WhatsApp adapter

I live in an iframe; the runtime talks to me via postMessage. I have no UI of my own beyond the log below — WhatsApp (real or emulated) is my actual interface. The WhatsApp Cloud API has no inline buttons and no message-edit endpoint, so my language choice is a numbered-reply menu, and the greeting always arrives as a new message. Conversation logic is shared with other platform adapters from ../shared/greet.js, unchanged.
