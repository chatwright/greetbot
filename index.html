Chatwright greetbot

A tiny language-onboarding bot — the first real Chatwright iframe demo bot. No framework, no build step, no dependencies: it's the proof that the iframe bot protocol is trivial to implement directly.

Platform bots

Conversation logic lives once, in shared/greet.js, entirely independent of any platform or protocol. Each row below is a thin adapter that translates one platform's native updates into that shared logic, and its returned intents back into that platform's own API calls.
