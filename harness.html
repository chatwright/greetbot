GreetBot manual test harness

This page plays the host side of the Chatwright iframe bot protocol handshake, by hand, so an adapter (telegram/ or whatsapp/, picked below) can be exercised without the real Chatwright runtime. It is a local dev tool only: a real host validates the bot's event.origin against the origin its CHATWRIGHT.md declares before handing over the MessagePort — this harness accepts any origin and only checks that the bot's declared platform matches the adapter selected below, logging a warning (not a hard failure) on mismatch. Never skip the origin check in a real host.
