GreetBot manual test harness

This page plays the host side of the Chatwright iframe bot protocol handshake, by hand, so telegram/ can be exercised without the real Chatwright runtime. It is a local dev tool only: a real host validates the bot's event.origin against the origin its CHATWRIGHT.md declares before handing over the MessagePort — this harness accepts any origin and simply echoes back whatever protocolVersion/ platform the bot announced. Never do that in a real host.
